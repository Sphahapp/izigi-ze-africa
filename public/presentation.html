Architecture (high‑level)

  • React 18 + Vite + TypeScript + Tailwind + shadcn/ui
  • Image Gen via Pollinations (URL-based), Vision via transformers.js (WASM)
  • Canvas composition for masking, inpainting, and PNG export
  • AI Builder: streaming code into sandboxed iframe, JSZip export
Private, fast, and web-native; easy to extend with more models.

Notable engineering work
